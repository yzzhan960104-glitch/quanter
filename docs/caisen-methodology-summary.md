## 2. 核心理念：滿足計算（目標價測量）【原著直接 · 公式已交叉验证】

**这是蔡森区别于"突破即买入"普通形态派的灵魂。**

- 【原著直接】蔡森不只識別形態，更用「滿足計算」科學測量漲幅目標（滿足點），作為止盈依據。
- 【原著直接 · 公式】（来自技术篇三 p017 正文 OCR + 鉅統/愛之味案例数值交叉验证）：

  ```
  颈线高度 H = 颈线价 − 谷底价
  第一波满足 = 颈线价 + H      （突破颈线后等额涨幅投影）
  第二波满足 = 第一波满足 + H   （整理再突破后，再等额累加）
  # 以此类推，可有多级满足点
  ```

- 【原著直接 · 案例验证】（技术篇三 p021-023，2001 鉅統日线）：
  - 谷底 36.7 → 颈线 59.5，颈线高度 H = 22.8
  - 第一波满足 = 59 + 22.8 = **81.8**（实测最高 79.18，接近）
  - 第二波满足 = 81.8 + 22.8 = **104.6**（实测最高 104.01，精准命中后盘头反转）
- 【原著直接 · 案例验证】（技术篇三 p021，2008-2009 愛之味日线）：
  - W 底颈线中位 7.6 − 谷底 4.65 = H = 2.95
  - 突破颈线（约 7.3）后目标 = 7.3 + 2.95 = **10.25**（实测满足 10.65，到位）
- 【原著直接 · 案例验证】（技术篇三 p017 消费股案例）：
  - 颈线中位 19 − 双底中位 14 = 颈线 5
  - 第一波满足 = 18 + 5 = **23**；第二波满足 = 23 + 5 = **28**
- 【原著直接】此法系统化应用于 W底、M头、破頭鍋、波段颈线、收敛三角形、旗形、底部区域等所有形态。

**对本项目的意义**：spec 第 6 节止盈"等距测量"应正名为「颈线满足计算」，并明确支持多级满足点（1倍/2倍，等额累加而非倍数相乘），这是蔡森方法的数学内核。`目标_n = 颈线价 + n × H`（n=1,2,...），不是 `颈线价 × n`。

## 3. 形態體系（多頭買進訊號）【原著直接 + 推断】

本项目为纯多头，聚焦蔡森的"多头买进讯号"形态族：

| 形态 | 含义 | 适用 | 来源 |
|---|---|---|---|
| **W 底（碗底翻）+ 颈线满足** | 双底 + 颈线突破，目标用颈线满足计算 | 核心 | 【原著直接】技术篇三 p018-026 |
| **破底翻** | 跌破底部区下缘颈线后翻升站回颈线，多头止稳买进讯号（主力低档进货刻意破底再拉升） | 进阶底部型 | 【原著直接】技术篇 p017 OCR |
| **破頭鍋**【推断】 | 突破前头部压力区（"锅"=前头部/套牢区），多头买进 | 突破型 | 【推断】 |
| **波段颈线满足** | 测量下个压力位，波段操作 | 趋势型 | 【原著直接】标题 |
| **收敛三角形底部（多头买进跳跃）** | 三角形收敛末端向上突破 | 波动收敛型 | 【原著直接】标题 |
| **底部区域计算（多头买进跳跃）** | 底部构筑完成后的买点 | 打底型 | 【原著直接】标题 |

### 破底翻详解【原著直接 · 技术篇 p017 OCR】
股价在盘跌一段进入筑底过程中，出现**跌破底部区下缘颈线，随后又翻升站回颈线**，属于破底翻结构。技术面出现破底翻时，大多是难得的**多头止稳买进讯号**。这种情况多是主力在低档筑底进货期，刻意或藉由利空随势破底，既然是主力进货，一旦破底便很容易拉升站回颈线。**破底翻是比颈线突破更进阶、更安全的底部布局方式，等待时间较长但获利空间更大**（技术篇 p017 註）。

### 碗底翻（W底）力道【原著直接 · 技术篇三 p018 OCR】
- W 底第二只脚（右脚）盘旋时出现碗底翻动作，**突破起涨关卡后碗底翻力道出现**
- 拉回时碗底翻的买点再度出现，**这个买点力道最强**
- 碗底翻的力道是 **2.5 倍**【OCR 辨识，案例具体值】

## 4. 打底理論（底部構築 ABC 波）【原著直接】

- 【原著直接】自序核心觀點："我在任何時候都會堅持『打底是穩健』...短線操作利潤是一時的，風險卻是永久的...從未聽過有任何人靠短線致富"。
- 【原著直接 · 实战篇二 p128 OCR】打底的標準範例就是 **A 波下跌、B 波反弹、C 波末跌**。
  - **A 波**：下跌波段（跌幅通常最大）
  - **B 波**：反弹波段（反弹幅度约 A 波的 1/3 至 1/2，反弹弱）【OCR 辨识比例，实现时可配置】
  - **C 波**：末跌波段（下跌幅度小于 A 波，缩量）
  - **打底完成判定**：**C 波不创新低（低点垫高）** = 打底完成
  - C 波完成后，待突破颈线即为多方转折点

**对本项目的意义**：PatternScreener 不能只识别"已成形的 W底静态四点"，应识别"打底过程"——即 C 波末跌/右底构筑的时序特征（C 波低点 > A 波低点），避免把下跌中继误判为 W底。这是蔡森区别于教科书 W底的关键：**强调过程与时序，而非静态四点**。

## 5. 多空轉折（書名核心）【原著直接 · 实战篇四 p132-133 OCR】

- 【原著直接】实战篇四「底部反轉＆多方轉折」是全书书名"多空转折"的落点章节。
- 【原著直接 · 实战篇四 p132 正文 OCR】**多方转折的精确定义**：

  > 多头市场的底部反转多方转折，**必须在打底完成后出现**。所谓的"打底"至少要经过 **ABC 三波**，也就是说下跌→反弹→再下跌，而**最后的一波下跌不能再创新低，也就是低点垫高**，这是底部反转多方转折的**必要条件**。多头市场的打底通常在 **26 周平均线之上**完成（26 周线是市场长期成本），26 周线是空头市场止跌转多的关键位置（图四之一）。

- 【原著直接 · 实战篇四 p132-133】**多方转折点确认条件（三重共振）**：
  1. **ABC 三波打底完成**：C 波低点垫高（不破 A 波低点）
  2. **26 周平均线确认**：
     - 多头市场：打底在 26 周线之上完成
     - 空头市场：打底在 26 周线之下完成，**当股价重新站回 26 周线之上，就是空头市场转多方的多方转折点**
  3. **带量长红/突破前波高点**：量能确认 + 突破前波高点
  - 三者成立 = **多方转折确立，为多头买进讯号**

- 【原著直接 · 实战篇四 p133 操作方式】多方转折确立后进场作多，**停损设于 C 波低点之下**（破 C 波低点即停损），目标看 W底颈线满足计算。

**对本项目的意义**（重大校准）：
- spec 的 PatternScreener 必须新增 **26 周均线（≈ 半年线/120 日均线）** 作为打底环境过滤——这是蔡森原著明确的数字，不是任意参数。
- 多方转折不是单一信号，而是 **ABC 打底 + 26 周线位置 + 带量突破** 三重共振。
- 停损位明确为 **C 波低点**（右脚），而非固定百分比。

## 6. 形態質量判別【原著直接标题 + 精读升级规则】

### 6.1 双底幅宽与张力（实战篇六 p136-137）【原著直接 · OCR】

- 【原著直接 · 实战篇六 p136 OCR】**双底（W底）张力核心规则**：
  - W 底有左脚与右脚，**右脚必须高于左脚**——这是 W 底形态的张力所在
  - **右脚价 > 左脚价**（右脚不能破左脚；破左脚就不是 W 底，是下跌中继）
  - **幅宽** = 两底之间的时间间距；幅宽越宽，代表主力进货时间越长，未来涨幅满足越大
  - **张力** = 右脚相对左脚的抬高度（右脚越高，张力越强）
  - 关系：**幅宽越宽 + 张力越强 → 涨幅满足越大**

- 【原著直接 · 实战篇八 p141 OCR】W 底满足计算公式（与技术篇三一致，见第 2 节），操作要点：
  - 突破颈线进场，停损设于颈线下缘（破颈线即停损）
  - 第一波满足附近可部分获利了结
  - 第二波满足附近再多单全数出场

**对本项目的意义**（重大校准）：
- W 底识别必须强制 **右脚价 > 左脚价**（右脚不能破左脚）——这是蔡森原著的硬规则，不是软偏好。右脚破左脚 = 下跌中继，直接否决。
- 形态质量评分应纳入 **幅宽（两底时间跨度）** 作为正向因子。

### 6.2 形態失敗（形態完成）（实战篇十 p144-145）【原著直接 · OCR】

- 【原著直接 · 实战篇十 p145 OCR】**形态失败定义与操作规则**：
  - **形态失败 = 颈线突破后，股价回落跌破颈线（假突破），形态完成，视为反向讯号**
  - 操作：突破颈线进场后，若股价回落跌破颈线，即判 定形态失败，**多单停损出场**，甚至可反向做空
  - 假突破判定：【OCR 辨识阈值】收盘跌破颈线约 3%、3 日内回落跌破——**这些具体数字为 OCR 辨识，实现时应作为可配置参数**（spec 的 `false_breakout_window` / `false_breakout_threshold`）

**对本项目的意义**：spec §5.4 假突破防御的蔡森原典依据已确认。假突破 = 跌破颈线 = 形态失败 = 反向信号，超窗即多单止损出场。

### 6.3 其他形态（仍部分推断，待后续精读）

- **標準型態 vs 反市場型態**：【推断】标准形态按教科书走；反市场形态是"该涨不涨/该跌不跌"的逆向信号。
- **3-3-3 型態 / 11/16 型態**：【推断】蔡森自创的数字化形态命名（可能指各波段的比例/长度关系），待精读。

## 7. 量價觀念【原著直接】

- 【原著直接】推荐序标题即"精準量價觀念，進出股市輕鬆抓買賣時機"——量价是蔡森方法的基础维度。

**对本项目的意义**：spec 量价配合（突破放量、右底缩量）是蔡森方法的正确方向，应强化为硬过滤条件。

## 8. 操作紀律【原著直接来自自序】

- "操作不靠急躁，行情是要等的"——耐心等待形态成熟。
- "打底是穩健"——拒绝短线频繁交易，波段操作。
- "投資要像農夫一樣，春耕、夏耘、秋收、冬藏"——尊重市场周期。
- "市場回檔反而是加碼的時機"——回踩买入（与 spec 回踩入场一致）。
