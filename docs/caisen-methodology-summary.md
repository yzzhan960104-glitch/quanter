## 2. 核心理念：滿足計算（目標價測量）【p017 圖直接 + 推断公式】

**这是蔡森区别于"突破即买入"普通形态派的灵魂。**

- 【原著直接】蔡森不只識別形態，更用「滿足計算」科學測量漲幅目標（滿足點），作為止盈依據。
- 【p017 图直接】W 底頸線滿足計算圖：標註了頸線、左右谷底、以及突破後向上投影的「滿足目標線」，含多個滿足點。
- 【推断公式】W 底頸線滿足 = `頸線價 + (頸線價 − 谷底價)`，即「頸線高度」向上等距投影；存在 1 倍滿足、2 倍滿足等多級目標（對應"幅寬與張力"）。
- 【原著直接】此法系統化應用於 W底、M頭、破頭鍋、波段頸線、收斂三角形、旗形、底部區域等所有形態。

**对本项目的意义**：spec 第 6 节止盈"等距测量"应正名为「颈线满足计算」，并明确支持多级满足点（1倍/2倍），这是蔡森方法的数学内核，而非随意等距。

## 3. 形態體系（多頭買進訊號）【原著直接 + 推断】

本项目为纯多头，聚焦蔡森的"多头买进讯号"形态族：

| 形态 | 含义 | 适用 |
|---|---|---|
| **W 底 + 颈线满足** | 双底 + 颈线突破，目标用颈线满足计算 | 核心 |
| **破頭鍋**【推断】 | 突破前头部压力区（"锅"=前头部/套牢区），多头买进 | 突破型 |
| **波段颈线满足** | 测量下个压力位，波段操作 | 趋势型 |
| **收敛三角形底部（多头买进跳跃）** | 三角形收敛末端向上突破 | 波动收敛型 |
| **底部区域计算（多头买进跳跃）** | 底部构筑完成后的买点 | 打底型 |

## 4. 打底理論（底部構築 ABC 波）【框架推断】

- 【原著直接】自序核心觀點："我在任何時候都會堅持『打底是穩健』...短線操作利潤是一時的，風險卻是永久的...從未聽過有任何人靠短線致富"。
- 【原著直接】實戰篇二「打底的標準範例：A/B/C 波」—— 蔡森用 A/B/C 三波描述底部構築過程。
- 【推断】A/B/C 波类似艾略特修正浪结构：A 波下跌、B 波反弹、C 波末跌（最终底），C 波结束才是真正的多方转折点。这比"识别一个静态 W 底"更注重**底部形成的过程与时序**。

**对本项目的意义**：PatternScreener 不能只识别"已成形的 W底静态四点"，应识别"打底过程"——即 C 波末跌/右底构筑的时序特征，避免把下跌中继误判为 W底。

## 5. 多空轉折（書名核心）【框架推断，待精读实战篇四】

- 【原著直接】实战篇四「底部反轉＆多方轉折」是全书书名"多空转折"的落点章节。
- 【推断】"多方转折点" = 下跌趋势结束、转为多头的关键确认点。结合"打底 ABC 波"，C 波末跌后出现的转折信号。
- 【推断】转折判定可能结合：颈线突破、量能确认、打底完成三重共振。

**待精读**：实战篇四正文，获取多方转折点的精确定义与判定规则。

## 6. 形態質量判別【原著直接标题 + 推断规则】

- **幅寬與張力**（实战篇六 双底 / 篇九 潜艇线）：【推断】形态的"宽度"（时间跨度）与"高度"（幅度）的比例关系，决定形态的"张力"（爆发力）。双底两底的价格关系（等高/右底略高）是质量关键。
- **標準型態 vs 反市場型態**：【推断】标准形态按教科书走；反市场形态是"该涨不涨/该跌不跌"的逆向信号。
- **型態失敗（形態完成）**（实战篇十）：【推断】形态未按预期方向突破（假突破），或突破后回落，蔡森将其视为"形态完成"的反向信号——即假突破后做反向操作。**这正是 spec 假突破防御的蔡森原典依据。**
- **3-3-3 型態 / 11/16 型態**：【推断】蔡森自创的数字化形态命名（可能指各波段的比例/长度关系），待精读。

## 7. 量價觀念【原著直接】

- 【原著直接】推荐序标题即"精準量價觀念，進出股市輕鬆抓買賣時機"——量价是蔡森方法的基础维度。
- 【推断】形态突破需量能确认（放量突破有效），缩量突破视为信号不足。

**对本项目的意义**：spec 量价配合（突破放量、右底缩量）是蔡森方法的正确方向，应强化为硬过滤条件。

## 8. 操作紀律【原著直接来自自序】

- "操作不靠急躁，行情是要等的"——耐心等待形态成熟。
- "打底是穩健"——拒绝短线频繁交易，波段操作。
- "投資要像農夫一樣，春耕、夏耘、秋收、冬藏"——尊重市场周期。
- "市場回檔反而是加碼的時機"——回踩买入（与 spec 回踩入场一致）。
